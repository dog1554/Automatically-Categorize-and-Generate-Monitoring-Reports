使用方式：
1. 把每一期成果表放在各期文件夹中，例如 001期、002期、059期。
2. 双击“运行自动分类监测统计.bat”，或运行：
   python 自动分类生成监测统计表.py
3. 程序会自动生成：
   *测次数统计表_自动生成.xlsx
   *监测次数统计表_自动生成明细.json

自动规则：
1.只统计可见工作表；隐藏工作表视为旧表或辅助表，不计入。
2.普通项目按工作表名自动分类：
  “地表沉降”“地表沉降(2)”“地表沉降（3）”都会归为“地表沉降”。
3.普通项目点数：主表 A 列“点号”去重。
4.普通项目次数：本周累计变量区内有实际数据的天数，多张同类表取最大值，不累加。
5.测斜项目：默认“工作表名像点号，且表内 A 列有深度”的工作表为测斜。
  例如 ZQT01、ZQT02、CX01 等会统一归到“测斜”。
6.测斜点数：可见点号工作表去重。
7.测斜次数：测斜表本周累计变量区内有实际数据的天数，多张表取最大值。
